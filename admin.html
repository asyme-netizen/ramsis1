الوضع الحالي: هذه النسخة تعمل مباشرة على GitHub Pages وتخزن البيانات محليًا داخل المتصفح تلقائيًا، لذلك ستشاهد اللوحة شغالة فورًا.

للتحويل إلى وضع فعلي على الإنترنت: أضف بيانات Supabase العامة داخل data.js في المتغيرين window.RAMSES_SUPABASE_URL و window.RAMSES_SUPABASE_ANON_KEY ثم فعّل الجداول والسياسات الموجودة في ملف الشرح.

الجداول المطلوبة: page_views و newsletter_subscribers و contact_messages.

المهم: لا تضع Service Role Key داخل GitHub Pages نهائيًا. استخدم فقط المفتاح العام anon.

الربط الفعلي

ربط Supabase والبيانات الحقيقية.

تم تجهيز الربط داخل الكود بالفعل. الملف supabase-config.js يحتوي على رابط مشروعك ومفتاح النشر العام.

المطلوب المتبقي مرة واحدة فقط: تشغيل ملف supabase-setup.sql داخل SQL Editor في Supabase حتى يتم إنشاء الجداول والسياسات.

الجداول المستخدمة: page_views و newsletter_subscribers و contact_messages و site_sections.

مهم: لا تضع أي مفتاح سري داخل GitHub Pages. هذه النسخة تستخدم المفتاح العام فقط.
